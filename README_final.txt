🚀 So startest du:

2. Öffne ein Terminal oder eine Konsole im Ordner
3. Führe aus:

✨ Funktionen:
- Mobilfreundliches Layout
- Mistdienst-Erinnerung (1 Tag vorher)
- Monatskalender mit hervorgehobenen Ereignissen
- Einsteller können Mistdienste eintragen, verschieben und löschen
- Admin kann Nutzer & Pferde anlegen
- Zahlungsstatus (z. B. „offen“) wird gespeichert
